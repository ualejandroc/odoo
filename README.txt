

Paso usar git en consola:

1. Instalar git
1.1 generar clave privada con putty en windows y ssh-keygen en linux
2. Crear carpeta con mismo nombre del proyecto y acceder a ella
3. git init
4. Crear o anadir archivo.
5. git add   // anade el archivo al control versiones local
6. git commit  // cambios en local
7. git commit -m "Descripcion de cambios en commit"    
8. git push origin master   // envia cambios a git pero a la seccion master si hubiera branch hay que especificar
